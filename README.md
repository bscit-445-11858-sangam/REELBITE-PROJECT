# ReelBite - Food Ordering Web App 🍔

## 📌 Description
ReelBite is a food ordering web application where users can explore food reels, add items to cart, place orders, and track deliveries.

## 🚀 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ✨ Features
- User Authentication
- Food Browsing (Reels Style)
- Add to Cart
- Order Placement
- Order Tracking

## 📂 Project Structure
- frontend/
- backend/

## ⚙️ How to Run
1. Clone repo
2. Install dependencies:
   npm install
3. Run backend:
   npm start
4. Run frontend:
   npm run dev
